Example API documentation version 0.1

http://example.com/0.1

/account

post /account

post Creates a new account

get put delete /account/{id}

get Get the account put Update the account delete Delete the account

post /account/login

post Login with email and password

post /account/forgot

post Sends an email to the user with a link to set a new password

get delete /account/session

get Gets the sessions delete Deletes the session, logging out the user

/conversations

get post /conversations

get Get a list of conversation for the current user post Create a new conversions. The currently logged in user doesn't need to be supplied in the members list, it's implied.

get put /conversations/{convId}

get Get a single conversation including its messages put Update a conversation (change members)

get post /conversations/{convId}/messages

get Get the messages for the conversation post Add a new message to a conversation

put delete /conversations/{convId}/messages/{messageId}

put Update the message delete Delete the message

/users

get post /users

get Get a list of all users post Creates a new user

get put delete /users/{userId}

get Get the details of a user including a list of groups he belongs to put Update a user delete Deletes a user

/groups

get post /groups

get Get a list of all the groups post Create a new group

get put delete /groups/{groupId}

get Get the details of a group, including the member list put Update the group, **optionally** supplying the new list of members (overwrites current list) delete Removes the group

post /groups/{groupId}/users

post Adds a user to a group

delete /groups/{groupId}/users/{userId}

delete Removes a user from a group
